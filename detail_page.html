프로젝트 A (웹사이트 리디자인)

2024년 · 역할: 프론트엔드 개발자

프로젝트 개요

프로젝트 A는 기존 웹사이트를 최신 디자인으로 전면 재구축한 작업입니다. 사용자 경험을 개선하고 반응형 디자인을 적용하여 다양한 기기에서 최적의 뷰를 제공합니다.

주요 기능

  • 반응형 레이아웃 구현 (Bootstrap Grid 활용)
  • 회원가입/로그인 페이지 개발
  • API를 연동한 실시간 데이터 로딩

사용 기술

  • HTML5, CSS3, JavaScript
  • Bootstrap 5 (그리드 및 컴포넌트)
  • jQuery, Node.js
